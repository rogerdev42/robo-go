üîê Authentication Required

Most endpoints require authentication. To test the API:

  1. First, use /auth/signup or /auth/signin to get a JWT token
  2. Click the "Authorize" button below and enter: Bearer YOUR_TOKEN_HERE
  3. Now you can test protected endpoints

üöÄ Quick Start

Base URL: http://localhost:8888/api

Content-Type: application/json

Example Authentication Flow:

  1. Register: POST /auth/signup with email, name, password
  2. Login: POST /auth/signin with email, password
  3. Use token: Add Authorization: Bearer <token> header to requests
